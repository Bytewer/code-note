# 5.2 类比推理核心考点
## 逻辑关系
+ 词项所代表的概念之间的集合关系
    - 全同
    - 包含
    - 交叉
    - 全异
+ 词项所代表的事件之间的条件关系
+ 词项所代表的的事件之间的因果关系

## 言语关系
+ 近义关系
+ 反义关系
+ 象征关系
    - 一个词语是另一个词语的象征义
+ 主谓结构
+ 动宾结构
+ 并列结构
+ 修饰关系

## 常识关系
+ 经验常识
+ 理论常识
